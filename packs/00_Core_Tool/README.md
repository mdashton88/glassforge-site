# Vehicle Forge — Core Reference Builds

**Price:** FREE (included with Vehicle Forge tool)
**Vehicles:** 64
**Wave:** 1 (Launch)
**Status:** Ready to ship

---

## Contents

64 generic and canon reference vehicles covering every locomotion type in the core tool. These are the vehicles GMs reach for when players steal a bus, commandeer a fishing trawler, or hop on a dirt bike. Organised by category:

**Civilian Ground (18):** Carriage, Model T, street bike, dirt bike, compact car, mid-size car, minivan, SUV, sports car, semi-truck, pickup truck, luxury sedan, city bus, panel van, ambulance, fire engine, police cruiser, stretch limo, ATV/quad, dump truck, bulldozer, snowmobile

**Military Ground — WWII (6):** Jeep, M4 Sherman, M5A1 Stuart, T-34/76, Panzer IVJ, King Tiger

**Military Ground — Modern (5):** M1A1 Abrams, M2 Bradley, T-72, T-80, BTR-70

**Military Ground — Future (2):** Hover Tank, Hover APC

**Military Ground — Generic (3):** Light Tank, MBT, APC

**Civilian Aircraft (5):** Biplane, Cessna, civilian helicopter, Learjet, Space Shuttle

**Military Aircraft — WWII (4):** B-17, Bf-109, P-51 Mustang, Spitfire

**Military Aircraft — Generic (3):** Jet fighter, attack helicopter, transport helicopter

**Civilian Aircraft — Other (3):** Airliner, cargo plane, ultralight, hang glider, hot air balloon

**Watercraft (7):** PT Boat, speedboat, yacht, fishing trawler, motorboat, rowboat, canoe, sailing ship

**Other (2):** Hovercraft, hover vehicles

## .vfx Dependencies

None. All weapons are from the core tool catalogue.

## One-Sheet Adventure

**Not applicable.** The Core Reference Builds ship as part of the free tool and do not include a standalone adventure. They serve as the foundation that all expansion packs build upon.
